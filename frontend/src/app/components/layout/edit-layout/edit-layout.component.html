How to Use the Layout Editor

Welcome to the Layout Editor!

Follow these steps to design your layout:

  1. Select a cell where you want to place a table and add all necessary data.
  2. To make the table bigger or to shape it as you please, click on adjacent cells to add them to the table.
  3. If you make a mistake, simply click on the cell to delete it.
  4. One area can be marked as the "Main Area", which will be displayed first when making a reservation.
  5. When you are done, press "Save" to save the area.
